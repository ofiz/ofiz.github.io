Advanced Security Implementation

Project Overview

Comprehensive ISP management system demonstrating enterprise-grade security practices with advanced cybersecurity implementations including multiple layers of protection against common web vulnerabilities.

Key Features

  • XSS Protection: Systematic HTML character encoding and input sanitization
  • SQL Injection Prevention: Parameterized queries using SQLModel ORM
  • HMAC-SHA256 Password Hashing: Cryptographic security with per-user 32-byte salts
  • Multi-Layer Security: Progressive account lockout protection (3 attempts, 15min lockout)
  • JWT Authentication: Secure token-based authentication with HMAC-SHA256 signatures
  • HTTP Security Headers: CSP, X-Frame-Options, X-Content-Type-Options
  • Audit Logging: Comprehensive UserEvent tracking with timestamps and IP addresses
  • Password Policy: YAML-based configuration with history tracking and blacklist

Technologies Used

  • Backend: FastAPI (async high-performance), Flask (UI services)
  • Database: MySQL 8.0 with SQLModel ORM for type-safe queries
  • Security: PyMySQL, Python-JOSE (JWT), HMAC-SHA256, SHA-1 for reset codes
  • Infrastructure: Docker, Docker Compose with health checks
  • Frontend: Jinja2 templates with automatic HTML escaping

Deep Learning GDP Forecasting

Advanced Economic Prediction

Additional Visualizations

Project Overview

State-of-the-art deep learning project for economic forecasting, implementing multiple advanced neural network architectures to predict GDP trends with high accuracy using temporal patterns in macroeconomic data.

Key Features

  • Transformers Architecture: Attention-based mechanism with Multi-Head Attention and Positional Encoding for long-term dependencies
  • TCN (Temporal Convolutional Networks): Dilated convolutions for hierarchical temporal feature extraction and fast training
  • WaveNet: Causal CNN with dilated convolutions for fine-grained temporal dynamics and local pattern capture
  • Multi-Model Ensemble: Combines predictions from multiple architectures for robust forecasting
  • Feature Engineering: Advanced processing of economic indicators and GDP historical data

Model Predictions

  • Transformers: GDP growth rate 0.7282% - Balanced trend estimation ideal for long-term analysis
  • TCN: GDP growth rate 1.2485% - Optimistic forecast leveraging temporal hierarchies
  • WaveNet: GDP growth rate 0.3866% - Conservative prediction for risk-averse strategies

Technologies Used

  • Frameworks: TensorFlow, PyTorch for flexible deep learning implementation
  • Data Processing: Pandas, NumPy for manipulation and feature engineering
  • Visualization: Matplotlib, Seaborn for loss curves and prediction charts
  • Time Series: Custom sequence modeling with sliding windows

Technical Achievements

  • Achieved superior forecasting accuracy compared to traditional ARIMA(AutoRegressive Integrated Moving Average - A time series forecasting model) models
  • Implemented custom loss functions optimized for economic metrics
  • Developed comprehensive model evaluation framework with MSE and MAE
  • Created actionable economic insights from deep learning predictions
  • Built ensemble approach combining multiple architecture strengths

Cost Tracker API - RESTful Expense Management

Real-Time Financial Management System

Video Demo

Project Overview

A scalable backend service for user expense management featuring advanced Computed Monthly Reporting and automated persistence. Built with Node.js and MongoDB Atlas for cloud-based deployment.

Key Features

  • Complete RESTful API: Full CRUD operations for expenses and users
  • Computed Pattern: Optimized caching for fast monthly reports grouped by category
  • MongoDB Atlas Integration: Cloud NoSQL database with data integrity
  • System Logging: Pino/Morgan for tracking API activity and debugging
  • User Management: Create users with validation and retrieve calculated totals
  • Monthly Reports: Generate reports with automatic caching for past months
  • Data Validation: Strict input validation with proper HTTP status codes
  • Cloud Deployment: Live API on Render platform with health monitoring

API Endpoints

  • POST /api/add: Creates new cost entry with automatic totalCost update
  • GET /api/report: Retrieves monthly expense report grouped by category
  • GET /api/users/:id: Gets user details with cumulative total costs
  • GET /api/about: Returns team members list

Technologies Used

  • Backend: Node.js & Express.js for server framework
  • Database: MongoDB Atlas (cloud NoSQL) with Mongoose ODM
  • Middleware: CORS for security, Morgan for HTTP logging
  • Logging: Pino for professional structured logging
  • Testing: Python unittest & requests for automated integration tests
  • Deployment: Render for cloud hosting

Technical Achievements

  • Implemented computed pattern for optimized monthly report generation
  • Created comprehensive test coverage with Python integration tests
  • Designed RESTful API following industry best practices
  • Built automatic timestamp assignment with backdated entry prevention
  • Deployed to production with continuous health monitoring
  • Implemented efficient data retrieval and processing with Mongoose
